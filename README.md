## Usage
firstly download the source:
```
$ git clone https://github.com/hasanpasha/cinemana
$ cd cinemana
```
Now you need to get the reqired packages:
```
$ pip install -r requirments.txt
```
To run the program:
```
$ python cinemana.py
```
